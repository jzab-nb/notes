三种形式：
内联式：直接写在开始标签里
嵌入式：写在head标签里
外部式：写在单独的.css文件里，在head里用<link href="文件路径" rel="stylesheet" type="text/css"/>标签引用rel和type属性是固定格式
优先性：内联式>嵌入式>外部式(一般情况)就近原则离元素越近越先显示
注释/* */
选择器{
          声明( 属性：值)
          }
1.标签选择器 直接写标签
2.类选择器     .类名          需要先用class=""定义类名
3.id选择器     #id名         需要先用id=""定义id
4.类选择器与标签选择器的区别
类选择器可以被多个标签使用如:<p class="e">e</p>
			<div class="e">e</p>
也可以为一个标签定义多个类如:<p class="ass we can"></p>
但标签选择器没有此功能
5.子选择器>用于选择指定标签元素的第一代子元素，如:ol>li
					   <ol>
					          <li></li>
					  </ol>
6.后代选择器 用空格选择指定标签的所有后代都在选择范围内
7.通用选择器 选中所有标签 用*选择
8.伪类选择器 给不存在的标签设置样式(标签的状态) 如: a:hover{color:red}就是给a标签设置当鼠标滑过时变为红色伪类选择器还有active（鼠标点击时）focus（鼠标点击后且没点击其他位置之前）
9.分组选择器 同时给多个元素设置样式h1,p就是同时给h1和p设置样式
10. .box1.box2 用来选择同时具有两个class的元素 <div classs="box1 box2"
11. + 选择紧贴在div元素之后的元素（同级关系）
